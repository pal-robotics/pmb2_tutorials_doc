.. _installation:

********************************
PMB-2 Installation tutorial ROS2
********************************


Purpose
#######

This tutorial shows how to install all the packages you need to simulate PMB-2 navigation and mapping.

Installation
############

Install all the ros official packages for pmb2.

.. code:: bash

   sudo apt update
   sudo apt install ros-foxy-pmb2*
   sudo apt install ros-foxy-pal-gazebo-worlds
